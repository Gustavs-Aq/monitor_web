# API de Monitoramento de Sites

API para monitorar disponibilidade de páginas web e tempo de resposta.

## Tecnologias
- Python
- Flask
- SQLite
- Requests

## Funcionalidades
- Cadastro de URLs
- Verificação de status HTTP
- Medição de tempo de resposta
- Histórico de verificações
- Relatório com tempo médio

## Como rodar

```bash
pip install flask requests
python app.py
