# API de Monitoramento de Sites

criação de uma api de monitoramento web que cadastra URLs e registrar verificações (status e tempo de resposta)

## Tecnologias
- Python
- Flask
- SQLite
- Requests

## Funcionalidades
- Cadastro de URLs
- Verificação de status HTTP
- Medição de tempo de resposta
- Histórico de verificações
- Relatório com tempo médio

## Como rodar

```bash
pip install flask requests
python app.py
